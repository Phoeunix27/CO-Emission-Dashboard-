# CO-Emission-Dashboard-
Interactive Multi-Page Dashboard for Exploring Global CO₂ Emissions
This Streamlit application allows users to explore and analyze global CO₂ emissions across countries and sectors using interactive visualizations. The dashboard provides a clean, modern interface with multiple pages, enabling both data exploration and sector-specific insights.

Features

Multi-page navigation: Home, Dataset Explorer, and CO₂ Sector Dashboard
Dataset upload: Load your own CO₂ emissions dataset (CSV) or use sample data

Interactive visualizations:
- Line charts for emission trends over time
- Bar charts for top emitting countries and sector totals
- Box plots for emission distribution
- Sunburst charts for hierarchical breakdowns
- Animated charts to visualize changes over time

Sidebar filters: Select country, sector, or both for dynamic analysis
Dashboard aesthetics: Modern layout with banners, color-coded charts, and clean metrics

Dataset
The dashboard is designed for the CO₂ Emission by Sector dataset from Kaggle:
CO₂ Emissions by Sector Dataset
Columns include: country, date, sector, value, and timestamp.

Navigate the pages using the sidebar and explore the visualizations.
